# Python Job Application Tracker

## Overview
A command-line Python application to track and manage job applications
in a structured and persistent way.

## Problem Statement
Job applications are often tracked manually in notes or spreadsheets,
making it hard to maintain status and history.

## Solution
This project provides a simple CLI tool to:
- Add job applications
- View stored applications
- Persist data using CSV files

## Tech Stack
- Python
- CSV file handling
